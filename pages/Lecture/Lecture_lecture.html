Lecture錄播系統

產品簡介

順應網絡化學習、及大規模網路免費公開課程（MOOC）等潮流，全新教育模式帶來學習革命的衝擊，因而雲端視頻錄播教學解決方案在未來數位學習領域佔有很重要的份量。

Cloudpe-Lecture 是一套全自動視頻錄播教學系統，在課堂上同步錄製講師與教材的聲音與影像。這套系統可以預定排程，自動錄製、上傳、發佈教材至 VOD 平台上，讓教學者能大量累積數位教材與檢視教學方式，而學習者則透過 VOD 平台，可不受時空限制觀看，進行自主式學習；系統管理者能輕易透過遠端監控作大量視頻管理與使用分析。

Cloudpe-Lecture 是大專院校數位教材錄製的最佳選擇，也是各培訓單位線上進修的最佳幫手，更是企業組織遠距培訓的最佳夥伴。


產品特色
  • 授課錄影全自動化

    系統可預定排程，全自動化多視訊串流錄製、上傳與發佈。

  • 遠端監控管理系統化

    大量錄製影音課程的管理與教學使用評量分析。

  • 網路直播

    任何人可隨時隨地在有連網的環境下進行即時網路直播活動，同時感受實體參與的樂趣。

  • 標籤索引與教材自動分段功能

    講師能輕鬆標記課程重點，而使用者能迅速地點播到想加強學習的課程影片片段。

  • 大規模同時佈建多間教室

    系統支援分散式運算，可進行大規模佈署。

  • API 整合服務

    透過 API 快速整合知名 LMS 與提供客製化服務。

  • 錄播完全行動化

    錄播設備採用小型化設計，方便攜帶，並可搭配手機作移動使用。

  • 跨平台與跨裝置的數位學習環境

    學習者可跨時空限制，利用 APP 登入 VOD 平台觀看教材影片。

  • 高品質低頻寬

    系統具備優異影像壓縮技術，同時滿足高解析影像 品質與頻寬節省的需求。

  • 強大檢索及搜尋功能

    具 OCR 文字辨識技術，視頻課程可用關鍵字作主題與內文搜尋，方便學習者快速尋找到想觀看的課程。

  • 影片剪輯功能

    課程影片可以簡易清除不需要的片段。

  • 播放平台介面彈性化

    使用者可依自我習慣選取不同畫面模式作自主學習。
