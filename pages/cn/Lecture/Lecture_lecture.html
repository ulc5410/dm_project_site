Lecture录播系统

产品简介

顺应网络化学习、及大规模网路免费公开课程（MOOC）等潮流，全新教育模式带来学习革命的冲击，因而云端视频录播教学解决方案在未来数位学习领域占有很重要的份量。

Cloudpe-Lecture 是一套全自动视频录播教学系统，在课堂上同步录制讲师与教材的声音与影像。这套系统可以预定排程，自动录制、上传、发布教材至 VOD 平台上，让教学者能大量累积数位教材与检视教学方式，而学习者则透过 VOD 平台，可不受时空限制观看，进行自主式学习；系统管理者能轻易透过远端监控作大量视频管理与使用分析。

Cloudpe-Lecture 是大专院校数位教材录制的最佳选择，也是各培训单位线上进修的最佳帮手，更是企业组织远距培训的最佳伙伴。


产品特色
  • 授课录影全自动化

    系统可预定排程，全自动化多视讯串流录制、上传与发布。

  • 远端监控管理系统化

    大量录制影音课程的管理与教学使用评量分析。

  • 网路直播

    任何人可随时随地在有连网的环境下进行即时网路直播活动，同时感受实体参与的乐趣。

  • 标签索引与教材自动分段功能

    讲师能轻松标记课程重点，而使用者能迅速地点播到想加强学习的课程影片片段。

  • 大规模同时布建多间教室

    系统支援分散式运算，可进行大规模布署。

  • API 整合服务

    透过 API 快速整合知名 LMS 与提供客制化服务。

  • 录播完全行动化

    录播设备采用小型化设计，方便携带，并可搭配手机作移动使用。

  • 跨平台与跨装置的数位学习环境

    学习者可跨时空限制，利用 APP 登入 VOD 平台观看教材影片。

  • 高品质低频宽

    系统具备优异影像压缩技术，同时满足高解析影像 品质与频宽节省的需求。

  • 强大检索及搜寻功能

    具 OCR 文字辨识技术，视频课程可用关键字作主题与内文搜寻，方便学习者快速寻找到想观看的课程。

  • 影片剪辑功能

    课程影片可以简易清除不需要的片段。

  • 播放平台介面弹性化

    使用者可依自我习惯选取不同画面模式作自主学习。
